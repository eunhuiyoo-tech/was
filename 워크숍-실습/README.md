# 📊 업무 대시보드 & 분석 프로젝트

Kim Secretary - 업무 효율화 대시보드 및 웹사이트 분석 도구

## 🌟 주요 기능

### 📋 업무 대시보드
- **할 일 목록**: 우선순위별 작업 관리
- **주간 일정**: 회의 및 약속 추적
- **프로젝트 진행률**: 실시간 진행 상황 모니터링
- **매출 요약**: 월간/누적 매출 현황

**접속:** `tasks/dashboard.html`

### 📝 회의록 관리
비즈니스 회의 기록 및 액션 아이템 추적

**접속:** `reports/meeting-result.html`

### 📊 매출 현황
인터랙티브 차트를 통한 매출 데이터 시각화

**접속:** `chart.html`

### 🔄 업무 프로세스
조직도 및 업무 흐름 다이어그램

**접속:** `diagram.svg`

### 📈 사이트 분석
SubTrackr 웹사이트 종합 분석 리포트

**접속:** `report.html`

## 🎨 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: 모던 그라데이션, 다크모드 지원
- **Features**: 반응형 디자인, 부드러운 애니메이션

## 🚀 시작하기

### 로컬 실행
```bash
# 간단하게 etc/index.html 열기
start etc/index.html
```

### 라이브 서버 (추천)
```bash
# VS Code Live Server 확장 사용
# 또는
python -m http.server 8000
# http://localhost:8000 에서 확인
```

## 📁 프로젝트 구조

```
워크숍-실습/
├── etc/
│   └── index.html           # 메인 진입점
├── tasks/
│   └── dashboard.html       # 업무 대시보드
├── reports/
│   └── meeting-result.html  # 회의록
├── chart.html               # 매출 현황
├── diagram.svg              # 업무 프로세스
├── report.html              # 사이트 분석
├── README.md
└── package.json
```

## 🎯 네비게이션

모든 페이지에서 **상단 탭 메뉴**를 통해 이동 가능:
- 📋 할 일 (대시보드)
- 📝 회의록
- 📊 매출 현황
- 🔄 업무 프로세스
- 📈 사이트 분석

## 🌙 다크모드

모든 페이지에서 우측 상단의 🌙 버튼으로 다크모드 활성화/비활성화 가능

## 🎨 특징

- ✨ 부드러운 그라데이션 배경
- 🎯 직관적인 카드 레이아웃
- 📱 모바일 반응형 디자인
- 🌙 라이트/다크모드 토글
- ⚡ 빠른 로딩 속도

## 📖 사용자 가이드

### 대시보드 사용법
1. `tasks/dashboard.html` 접속
2. 좌측 상단 "← 메인"으로 메인 페이지 이동 가능
3. 상단 탭에서 원하는 페이지로 이동

### 다크모드 설정
- 우측 상단 🌙 버튼 클릭
- 설정이 로컬스토리지에 저장됨
